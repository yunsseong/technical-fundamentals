# 호스팅이란?
호스팅은 웹사이트나 서비스를 인터넷에 공개하기 위해 서버 공간과 자원을 빌려주는 서비스입니다.
개인이 직접 서버를 구축하지 않고도 웹 사이트를 운영할 수 있게 해주며,
대표적으로 공유 호스팅, VPS, 클라우드 호스팅 등이 있습니다.

## 종류
1. 공유 호스팅
   - 여러 사용자가 하나의 물리적 서버 자원을 공유하는 형태
2. VPS (Virtual Private Server, 가상 사설 서버)
   - 물리 서버 하나를 여러 개의 가상 서버로 분할하여 제공
3. 전용 서버 호스팅 (Dedicated Hosting)
   - 물리 서버 한 대를 단독으로 사용하는 방식
4. 클라우드 호스팅 (Cloud Hosting)
